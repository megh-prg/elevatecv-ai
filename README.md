# 🚀 ElevateCV AI

ElevateCV AI is a smart resume optimization tool that helps job seekers tailor their resumes to specific job descriptions. It analyzes your resume, checks ATS compatibility, and suggests improvements to boost your chances of getting shortlisted.

## ✨ Features
- ATS score based on job description  
- AI-powered resume analysis  
- Keyword matching & suggestions  
- Quick and automated feedback  

## 🛠️ Tech Stack
- Backend: Python (FastAPI)  
- AI/NLP: OpenAI / Gemini API  
- Tools: Git, REST APIs  

## ⚙️ How It Works
1. Upload your resume  
2. Paste the job description  
3. Get ATS score + improvement suggestions
